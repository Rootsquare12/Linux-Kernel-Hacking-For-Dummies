# 0-0. 환영합니다!

안녕하세요! 『Linux Kernel Hacking for Dummies』를 집필하게 된 Rootsquare입니다.

본 시리즈는 세계적으로 쓰이는 오픈 소스 운영체제인 리눅스(Linux), 그중에서도 운영체제의 심장부인 **커널(Kernel)**을 이해하고 직접 해킹해 보는 입문서입니다.

커널이 가질 수 있는 취약점을 분석하고 공격 원리를 배우며, 나아가 현대 운영체제에 적용된 보안 장치(Mitigation)들을 어떻게 우회할 수 있는지 학습합니다. 후반부에는 실존했던 취약점(CVE)들을 직접 실습하며 실전 감각도 익혀볼 예정입니다!

## ❓ 강의를 집필하게 된 계기가 있나요?

그동안 운영체제 커널 해킹은 세계적으로 꾸준히 연구되어 온 중요한 분야지만, 안타깝게도 국내에는 이를 단계별로 배울 수 있는 학습 자료가 극히 드물었습니다. 자료의 부족과 높은 난이도라는 거대한 진입 장벽 때문에 포기하는 분들을 많이 보았고, 이 문제를 조금이나마 해결해 보고자 본 프로젝트를 시작하게 되었습니다.

저 역시 처음 커널 해킹의 문을 두드릴 때 느꼈던 막막함과 답답함을 누구보다 잘 기억하고 있습니다. 초보자의 시선에서, 처음 공부할 때 가장 헷갈리고 어려웠던 부분들을 짚어가며 여러분과 함께 성장하는 마음으로 집필하겠습니다. 자, 그럼 시작해 볼까요?
